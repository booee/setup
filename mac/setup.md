# Mac Setup

## Installs

Use [Homebrew](http://brew.sh)

**Installing Homebrew**

`ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"`

Note: Install the terminal CLI tools

### Tools & Software (via Homebrew)

- google chrome
- node.js
- subversion (for v1.8)
- git
- bash-completion
- cask
    - shuttle
    - java (oracle)
    - atom text editor
    - slack
    - android-sdk
    - android studio
    - eclipse
